# SEO Blog Guide Template System
## 50 Guide Framework for Calculator Integration

This document provides the **complete template** and **prompt framework** for generating 50 SEO-optimized articles that drive traffic to your calculators.

---

## Quick Start: How to Use This System

### The Template Structure (Every Guide)
```
1. HOOK/INTRO (100-150 words)
   - Problem statement
   - Why the reader cares
   - Calculator CTA (soft)

2. MAIN CONTENT (5-7 sections, 1500+ words total)
   - Section 1: Core Concept Explained
   - Section 2: How It Works / Formula
   - Section 3: Step-by-Step Guide
   - Section 4: Real Examples
   - Section 5: Common Mistakes
   - Section 6: Advanced Tips
   - Section 7: Applications/Benefits

3. FAQ SECTION (4-6 questions)
   - Address common misconceptions
   - Link back to calculator feature

4. CTA BOXES (3-5 total)
   - Soft CTAs (throughout content)
   - Hard CTAs (gradient boxes at strategic points)
   - Link directly to calculator route

5. SEO METADATA
   - Meta Title (60 chars)
   - Meta Description (160 chars)
   - Keywords (8-10 variations)

6. FOOTER
   - SEO info box
   - Back link to blog
```

## Writing Style Guide

### Tone: Approachable Expert
- ✅ "Most people get this wrong" (relatable)
- ❌ "As per industry standards" (robotic)

### Sentence Structure: Simple
- ✅ "You need two numbers"
- ❌ "Requisite data includes two variables"

### Examples: Always Concrete
- ✅ "$100 at 20% off = $80"
- ❌ "Extract percentage value from principal amount"

### Pain Points: Lead with them
- ✅ Open with "Why restaurants make you feel dumb about tips"
- ❌ Open with "History of tipping culture"

### Real Numbers: Use them everywhere
- ✅ "Save $250-900/year"
- ❌ "Save significant money"

## Content Generation Workflow

### For Each Article:

1. **Copy the template** from above
2. **Fill in the blanks**:
   - Route (URL path)
   - Calculator link
   - Real examples with numbers
   - FAQ questions specific to topic
   - Keywords

3. **Write sections**:
   - Hook (100-150 words)
   - Main sections (1500+ total)
   - FAQ (4-6 questions)
   - CTA boxes (3-5)

4. **Add metadata box** with SEO info

5. **Create file at**: `app/seo-blog/[category]/[guide-name]/page.tsx`

6. **Link in blog page** to new guide

7. **Build and deploy** with `npm run build && vercel --prod`
